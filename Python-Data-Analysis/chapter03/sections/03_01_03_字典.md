#### 3.1.3 字典

- **dict**：键值对（hash map / 关联数组）；`del` 或 `pop` 删除。
- **`collections.defaultdict`**：为新键提供默认值，简化分组累加等逻辑。
- **可散列键**：键须不可变（int、float、str、元组等）；`hash()` 可检测；**列表不能作键**（需转元组）。
