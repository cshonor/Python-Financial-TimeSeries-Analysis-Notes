#### 3.2.6 错误和异常处理

- 脏数据常触发 `ValueError`、`TypeError` 等。
- **`try` / `except`**：可指定异常类型或元组捕获多种异常。
- **`else`**：仅 `try` 成功时执行；**`finally`**：无论是否异常都执行（清理资源）。
- IPython/Jupyter 异常栈更易调试。
