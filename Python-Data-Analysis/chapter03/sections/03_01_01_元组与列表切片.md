#### 3.1.1 元组与列表切片

- **元组（tuple）**：长度固定、**不可变**；`tuple()` 或逗号创建；索引从 0 开始。
- **切片 `[start:stop]`**：含 `start`，**不含** `stop`；可省略起止；负索引从后往前；`step` 可间隔取值，`[::-1]` 翻转。
- **与 NumPy/pandas 的关系**：`loc`/`iloc`、数组切片都建立在本节索引直觉之上。
