### 3.3 文件和操作系统

- **`open(path, mode)`**：`r`/`w`/`x`/`a`；`b` 二进制；`t` 文本（默认 Unicode 解码）。
- **`read` / `write` / `writelines`**；`seek` / `tell` 定位。
- **推荐**：`with open(..., encoding="utf-8") as f:` 自动关闭。
- 后续虽多用 `pandas.read_csv`，但理解基础 I/O 与编码对日志、非标准数据仍必要。
