### 2.1 Python 解释器

- Python 是 **解释型** 语言，解释器一次执行一条语句。
- 终端输入 `python` 启动标准解释器，提示符 `>>>`。
- 数据分析领域更常用 **IPython / Jupyter**，而非裸 `python` REPL。
