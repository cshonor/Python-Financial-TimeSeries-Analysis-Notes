#### 2.2.2 运行 Jupyter Notebook

- `jupyter notebook` 或 `jupyter lab` 启动本地 Web 服务。
- 笔记本文件 **`.ipynb`**，含代码、输出与 Markdown。
- 关浏览器标签 **不会** 结束内核；需 **Kernel → Shutdown** 或 Close and Halt。
