### 2.2 IPython 基础

IPython 是数据分析的 **默认交互层**：既可 **`ipython` 命令行**写/跑代码，也是 **Jupyter 的 Python 内核**。底层执行仍靠 **CPython**（见 [2.1](./02_01_Python_解释器.md)）。

| 2.2.2 | 运行 Jupyter Notebook | [02_02_02](./02_02_02_运行_Jupyter_Notebook.md)（**IPython ↔ Jupyter**、能否写代码） |

| 2.2.4 | 自省 | [02_02_04](./02_02_04_自省.md) |

**一句话**：IPython 能写/跑交互代码 → 会话 = **内存中的临时巨型脚本**；CLI 用 `ipython`，Web 用 Jupyter（同一内核内存）→ `%run` / `%save` / `?`。
