#### 2.3.2 标量类型

- 主要标量：`None`、`str`、`bytes`、`float`、`bool`、`int`。
- **`/`** 真除法得 float；**`//`** 地板除。
- **`datetime`**：`strftime` / `strptime`、`timedelta`；量化中日期解析的基础。
- **`None`** 单例，用 **`is None`** 判断。
