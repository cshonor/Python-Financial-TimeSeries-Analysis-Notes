#### 2.3.3 控制流

- **`if` / `elif` / `else`**；`and`/`or` 短路；支持链式比较 `a < b < c`。
- **`for`**：迭代集合；`continue` / `break`；**拆包** `for a, b in pairs`。
- **`while`**：条件循环。
- **`pass`**：占位。
- **`range(start, stop, step)`**：不含 stop；省内存，常配合索引。
