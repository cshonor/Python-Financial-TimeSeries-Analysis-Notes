#### 8.1.3 使用 DataFrame 的列进行索引

- **`set_index(cols)`**：列 → 行索引；默认从数据中移除列，`drop=False` 可保留。
- **`reset_index()`**：索引层级 → 普通列，恢复默认整数索引。
