#### 8.2.2 根据索引合并

- **`left_index=True` / `right_index=True`**：以索引为键；层次化索引等价多键合并。
- **`df.join(other)`**：按索引合并，默认 **left join**；要求列名不重叠；可传入 DataFrame 列表多表拼接。
