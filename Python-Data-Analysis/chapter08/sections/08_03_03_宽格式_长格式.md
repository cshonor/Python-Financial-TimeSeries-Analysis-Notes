#### 8.3.3 宽格式 → 长格式（melt）

- **`pd.melt(df, id_vars=..., value_vars=..., var_name=..., value_name=...)`**
- `id_vars`：保持不动的键列；其余列折叠为 `variable` / `value`（列名可自定义）。
