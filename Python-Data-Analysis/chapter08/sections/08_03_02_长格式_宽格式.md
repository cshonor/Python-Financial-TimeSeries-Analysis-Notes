#### 8.3.2 长格式 → 宽格式（pivot）

- 长表：每行一次观测（如 date, item, value）。
- **`df.pivot(index=..., columns=..., values=...)`**；省略 `values` 时剩余列都作值列 → 层次化列索引。
- 等价于 **`set_index` + `unstack`**；要求 (index, columns) 组合唯一，否则用 **`pivot_table` + 聚合**。
