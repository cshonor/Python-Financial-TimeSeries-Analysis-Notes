#### 8.3.1 使用层次化索引进行重塑

- **`stack()`**：列 → 行（DataFrame → 带 MultiIndex 的 Series）。
- **`unstack()`**：行 → 列（stack 的逆）；默认操作**最内层**，可指定 `level`。
- `unstack` 可能引入 NA；`stack` 默认丢弃 NA，整齐数据上二者常可逆。
