#### 12.2.2 分类数据和 Patsy

- 非数值列 → **虚拟变量**；有截距时去掉一个基准水平防共线。
- 无截距（`+ 0`）时保留全部哑变量列。
- **`C(列名)`** 强制按分类处理数值编码列。
- **`key1:key2`** 生成交互项（ANOVA 常用）。
