#### 12.2.1 用 Patsy 公式进行数据转换

- 公式内可写 Python 函数：`np.log(np.abs(x1) + 1)`。
- 内置 **`standardize`**、**`center`**（有状态：须用训练集统计量变换新数据）。
- 真算术相加用 **`I(x0 + x1)`** 封装。
- 预测新样本：**`patsy.build_design_matrices([X.design_info], new_data)`** 复用训练时的转换状态。
