## 一、书本原文核心知识点提炼

### 4.4.1 `np.where(cond, x, y)` — 向量化三元逻辑

**作用**：替代元素级 `if/else`，**一次对整个数组**做条件选择（无循环）。

- 语法：`np.where(条件, 满足时的值, 不满足时的值)`
- `x`、`y` 可与 `cond` 广播

```python
import numpy as np

arr = np.array([1, 2, 3, 4, 5])
res = np.where(arr > 3, arr, 0)          # [0 0 0 4 5]

# 多条件
np.where((arr > 2) & (arr < 5), 99, arr)
```

**仅取索引**：`np.where(cond)` 无第二、三参数时，返回满足条件的 **坐标元组**（与 `arr.nonzero()` 类似）。

### 4.4.2 `mean` / `sum` / `std` + `axis`；`cumsum` / `cumprod`

#### 1）聚合函数（压缩维度）

| API | 作用 |
|-----|------|
| `arr.sum()` / `np.sum(arr)` | 求和 |
| `arr.mean()` | 均值 |
| `arr.std()` | 标准差（默认总体 ddof=0） |
| `arr.max()` / `arr.min()` | 最大 / 最小 |

**`axis` 参数**（二维记忆）：

| axis | 方向 | 含义 |
|------|------|------|
| **`axis=0`** | 沿行向下 | **按列**聚合（每列一个结果） |
| **`axis=1`** | 沿列向右 | **按行**聚合（每行一个结果） |

```python
arr2d = np.array([[1, 2, 3], [4, 5, 6]])

arr2d.sum()           # 21（全体）
arr2d.sum(axis=0)     # [5, 7, 9]  列和
arr2d.sum(axis=1)     # [6, 15]    行和
```

#### 2）累计函数（不压缩维度）

| API | 作用 |
|-----|------|
| `cumsum` | **累计和**（逐步累加） |
| `cumprod` | **累计积**（逐步相乘） |

```python
arr = np.array([1, 2, 3, 4])
arr.cumsum()          # [1, 3, 6, 10]
arr.cumprod()         # [1, 2, 6, 24]

arr2d.cumsum(axis=0)  # 按列累计
```

### 4.4.3 布尔数组：`sum` 计数；`any` / `all`

#### 1）布尔数组

比较运算 `> < == !=` 与逻辑 `& | ~` 得到 **同形状** 布尔数组（True/False）。

#### 2）`sum()` → 统计 True 个数

True=1、False=0 → **`mask.sum()` = True 的个数**。

```python
arr = np.array([1, 2, 3, 4, 5])
mask = arr > 3
mask.sum()    # 2（4、5）
```

#### 3）`any()` / `all()`

| API | 含义 |
|-----|------|
| `mask.any()` | **至少一个** True → True |
| `mask.all()` | **全部** True → True |

```python
mask.any()    # True
mask.all()    # False
```

---

### 4.4.4 `arr.sort()` 就地；`np.sort(arr)` 返回副本

| 写法 | 行为 | 原数组 |
|------|------|--------|
| **`arr.sort()`** | **就地排序** | ✅ **被修改**，返回 `None` |
| **`np.sort(arr)`** | 返回排序结果 | ❌ **不变**（副本） |

```python
arr = np.array([3, 1, 4, 2])
arr.sort()                    # arr → [1, 2, 3, 4]

arr = np.array([3, 1, 4, 2])
arr2 = np.sort(arr)           # arr 不变，arr2 = [1, 2, 3, 4]
```

二维：`arr2d.sort(axis=1)` 按行排序（就地）。

**找序号不排序**：`arr.argsort()` 返回排序后的 **索引**（见 [`10_numpy_search_sort.md`](../../code/numpy/10_numpy_search_sort.md)）。

### 4.4.5 `unique`、`in1d`、集合运算

#### 1）`np.unique(arr)`：去重 + 排序

```python
arr = np.array([2, 1, 3, 2, 1, 3, 3])

np.unique(arr)                              # [1, 2, 3]
vals, cnts = np.unique(arr, return_counts=True)
# vals: [1,2,3], cnts: [2,2,3]
```

#### 2）`np.in1d(a, b)`：a 的元素是否在 b 中

返回 **长度同 a** 的布尔数组。

```python
a = np.array([1, 2, 3, 4, 5])
b = np.array([2, 4, 6])
np.in1d(a, b)    # [False, True, False, True, False]
```

> NumPy 2.x 推荐等价写法：**`np.isin(a, b)`**（语义相同，新代码优先用 `isin`）。

#### 3）其他集合运算

| API | 作用 |
|-----|------|
| `np.intersect1d(a, b)` | 交集（已排序） |
| `np.union1d(a, b)` | 并集（已排序） |
| `np.setdiff1d(a, b)` | 差集 a−b（已排序） |

## 二、一页式速查清单（易错点对比）

### API 速查

| 场景 | API |
|------|-----|
| 条件赋值 | `np.where(cond, x, y)` |
| 按列聚合 | `arr.mean(axis=0)` |
| 按行聚合 | `arr.sum(axis=1)` |
| 累计净值 | `np.cumprod(1 + rets)` |
| True 计数 | `(arr > 0).sum()` |
| 是否存在 | `mask.any()` |
| 是否全满足 | `mask.all()` |
| 就地排序 | `arr.sort()` |
| 副本排序 | `np.sort(arr)` |
| 去重计数 | `np.unique(arr, return_counts=True)` |
| 成员检测 | `np.isin(a, b)` / `np.in1d(a, b)` |

### 易错点对比

| 易错 | 正确 |
|------|------|
| `arr.sort()` 以为有返回值 | 返回 **`None`**，原数组已变 |
| `axis=0` 记成「按行」 | **`axis=0` = 按列**（沿行方向压扁） |
| `where` 多条件不用括号 | `(a>2) & (a<5)` |
| 用循环做条件赋值 | 改 **`np.where`** 或布尔掩码赋值 |
| 误改排序后的原数据 | 需要保留原序时用 **`np.sort`** |

### 必背口诀

> **where 向量化三元，mean sum std 加 axis；  
> 布尔 sum 计 True，any all 判真假；  
> sort 就地、np.sort 副本；unique 去重、in1d/isin 成员。**

## 三、演示：量化向量化片段

```python
import numpy as np

rets = np.array([0.01, -0.02, 0.015, -0.005, 0.03])

# 信号：涨则 1，跌则 -1，否则 0
signal = np.where(rets > 0, 1, np.where(rets < 0, -1, 0))

# 累计净值曲线
nav = np.cumprod(1 + rets)

# 多少天收益为正
(rets > 0).sum()

# 截面因子矩阵 (n_stock, n_day) 按股票求均值
```

## 四、【量化专属改造】

1. **持仓信号**：`np.where(ma_short > ma_long, 1, 0)` 替代 for 循环；与 pandas `shift` 配合防未来函数（第 11 章）。
2. **净值曲线**：`np.cumprod(1 + rets)` 是回测必备；对齐 `cumsum` 对数收益近似。
3. **axis 截面/时序**：因子矩阵 `(n_stock, n_factor)` → `mean(axis=0)` 各因子截面均值；`axis=1` 各股均值。
4. **`sort` 陷阱**：对共享内存切片 `.sort()` 会改原数组 → 排序前 **`.copy()`** 或用 `np.sort`。
5. **unique 统计**：统计行业/板块编码出现次数 → `return_counts=True`。

## 五、与 statsmodels / pandas 的衔接

- OLS 残差分析：`resid.std()`、`(np.abs(resid) > 2*sigma).sum()` 向量化计数异常点。
- pandas **大部分聚合**底层即 NumPy；理解 `axis` 后读 `groupby().mean()` 更清晰。
- `pd.Series.sort_values()` ≈ 对 values 的排序，但索引对齐由 pandas 管。
