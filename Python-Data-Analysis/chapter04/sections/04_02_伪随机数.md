# 4.2 伪随机数（numpy.random）

> 配套脚本：[`../../code/numpy/17_numpy_random_module.py`](../../code/numpy/17_numpy_random_module.py)、[`14_numpy_routines.py`](../../code/numpy/14_numpy_routines.py)

## 核心一句话

NumPy 的伪随机数基于 **确定性算法**，由 **种子（seed）** 决定序列：**同种子 → 同随机数**；可复现、批量生成比 Python 内置 `random` 快很多。

---

## 一、书本原文核心知识点提炼

### 4.2.1 新旧 API：推荐用 `default_rng`

#### 旧版（不推荐，全局状态）

```python
import numpy as np

np.random.seed(42)           # 全局种子，影响整个 np.random
x = np.random.randn(1000)    # 标准正态
```

| 缺点 | 说明 |
|------|------|
| **全局共享状态** | 一处改 seed，处处受影响 |
| **多库/多线程干扰** | 与其他模块争用同一 RNG 状态 |
| **不可控** | 并行、回测多策略时难隔离 |

#### 新版（推荐，隔离生成器）

```python
import numpy as np

rng = np.random.default_rng(seed=42)   # 独立 Generator
x = rng.standard_normal(1000)          # 标准正态
```

| 优点 | 说明 |
|------|------|
| 算法 | 默认 **PCG64**，速度与随机性更好 |
| **隔离** | 每个 `rng` 状态独立，适合并行、多实验 |
| **可复现** | 固定 `seed=整数`，结果永远一样 |

---

### 4.2.2 常用随机数生成（新版 API）

#### 1）均匀分布 [0, 1)

```python
rng.random(size=5)           # 一维
rng.random(size=(2, 3))      # 二维
```

#### 2）标准正态 N(0, 1)

```python
rng.standard_normal(5)
rng.standard_normal((2, 2))
```

#### 3）整数随机数（离散均匀）

```python
rng.integers(low=0, high=10, size=5)   # [0, 10) 整数
```

#### 4）其他常用分布

```python
rng.normal(loc=5, scale=2, size=5)      # 正态 N(5, 2²)
rng.uniform(low=1, high=5, size=5)      # 连续均匀 [1, 5)
rng.binomial(n=10, p=0.5, size=5)       # 二项分布
# 另有 exponential, poisson, gamma 等 → rng.exponential(...)
```

#### 5）抽样与打乱（量化常用）

```python
rng.choice(arr, size=10, replace=False)  # 不放回抽样
rng.permutation(n_stocks)                # 随机排列
rng.shuffle(arr)                         # 原地打乱
```

---

### 4.2.3 与 Python 内置 `random` 对比

| | `random` 模块 | `np.random.default_rng` |
|---|---------------|------------------------|
| **速度** | 标量为主，慢 | **批量数组**，快几十倍 |
| **维度** | 标量 | **直接多维** `size=(n, m)` |
| **可复现** | `random.seed()` | `default_rng(seed)`，**隔离更好** |
| **分布** | 较少 | **正态、Gamma、泊松、二项** 等统计分布全 |

---

### 4.2.4 书中重点总结（必背）

1. **伪随机 = 确定算法 + 种子**；同种子 → 同序列（非真随机）。
2. **优先 `np.random.default_rng(seed)`**：隔离状态、可复现、更快。
3. **批量生成数组**，勿用 Python 循环 + `random.random()`。
4. 常用 trio：`rng.random()`、`rng.standard_normal()`、`rng.integers()`。

## 二、一页式速查清单（新旧 API + 易错点）

### 新旧 API 对照表

| 用途 | 旧版（全局，不推荐） | 新版（推荐） |
|------|---------------------|--------------|
| 设种子 | `np.random.seed(42)` | `rng = np.random.default_rng(42)` |
| [0,1) 均匀 | `np.random.rand(5)` | `rng.random(5)` |
| 标准正态 | `np.random.randn(5)` | `rng.standard_normal(5)` |
| 一般正态 | `np.random.normal(μ, σ, size)` | `rng.normal(loc=μ, scale=σ, size=...)` |
| 整数 | `np.random.randint(0, 10, size=5)` | `rng.integers(0, 10, size=5)` |
| 打乱 | `np.random.shuffle(a)` | `rng.shuffle(a)` |

### 易错点

| 易错 | 正确做法 |
|------|----------|
| 回测每次结果不同 | 开头 **`rng = default_rng(42)`**，全文用同一 `rng` |
| 多策略共用一个全局 seed | **每个实验独立 `rng`**，勿依赖 `np.random.seed` |
| 需要「真随机」 | 伪随机只能 **可复现**；/crypto 级随机另用 `secrets` |
| 旧教程 `randn`/`rand` | 新代码映射到 `standard_normal`/`random` |

### 记忆口诀

> **default_rng 要隔离，同 seed 可复现；  
> random 均匀 standard_normal 正态，integers 离散快批量。**

## 三、演示：可复现蒙特卡洛雏形

```python
import numpy as np

rng = np.random.default_rng(42)

# 模拟 1000 条 252 日收益路径（随机漫步前置）
n_paths, n_days = 1000, 252
steps = rng.standard_normal((n_paths, n_days)) * 0.01   # 日收益 ~ N(0, 1%)
paths = np.cumsum(steps, axis=1)                        # 沿时间轴累计

# 再次同样 seed → 完全一致
rng2 = np.random.default_rng(42)
assert np.allclose(steps, rng2.standard_normal((n_paths, n_days)) * 0.01)
```

完整示例见 [`../../code/numpy/17_numpy_random_module.py`](../../code/numpy/17_numpy_random_module.py)。

## 四、【量化专属改造】

1. **回测可复现**：策略脚本顶部 `RNG = np.random.default_rng(42)`；bootstrap、随机子样本共用或按需分叉。
2. **随机漫步 / 期权 MC**：批量 `standard_normal((n_sim, n_step))` + `cumsum`（衔接 [4.7 随机漫步](./04_07_随机漫步.md)）。
3. **训练/测试划分**：`rng.permutation(n)` 打乱索引；时序数据 **勿随机 shuffle**，用时间切分（见第 11 章）。
4. **与 pandas 采样**：`df.sample(frac=0.8, random_state=42)` 内部也依赖伪随机；与 NumPy seed **独立**，勿混用期望一致。

---

## 五、与 statsmodels / 建模的衔接

- **Bootstrap 标准误**：用 `rng.choice` 或重采样索引，固定 seed 保证论文/报告可复现。
- **模拟 p 值 / 置换检验**：`rng.permutation` 生成零假设下的标签排列。
- **残差模拟**：`rng.normal(0, sigma_hat, size=n)` 生成扰动项做敏感性分析。
