## 核心一句话

**元素乘（`*`）逐元素；矩阵乘（`@` / `dot`）按线性代数规则；复杂分解与方程组用 `np.linalg`。**

---

## 一、两种乘法（必背区分）

### 1）元素乘（Hadamard 积）：`*`

- **逐元素相乘**（ufunc），要求 **shape 相同**（或可广播）
- **不是**矩阵乘法

```python
import numpy as np

A = np.array([[1, 2], [3, 4]])
B = np.array([[5, 6], [7, 8]])

A * B
# [[ 5 12]
#  [21 32]]
```

### 2）矩阵乘法：`@` 或 `np.dot`

- 线性代数规则：**(m×n) @ (n×p) = (m×p)**
- 内维必须匹配：`A` 的列数 = `B` 的行数
- **`A @ B`** 等价 **`np.dot(A, B)`**（二维时）

```python
A @ B
# [[19 22]
#  [43 50]]

np.dot(A, B)   # 同上
```

| | `*` | `@` / `dot` |
|---|-----|-------------|
| 含义 | Hadamard 逐元素 | 矩阵乘 |
| shape | 通常相同 | 内维对齐 |
| 与 ufunc | 即 `multiply` | 非 ufunc，线代运算 |

---

## 二、`np.linalg` 核心函数

### 1）逆矩阵：`np.linalg.inv(A)`

- 仅 **满秩方阵** 可逆
- 验证：`A @ inv(A) ≈ I`（浮点误差用 `np.allclose`）

```python
A = np.array([[1., 2.], [3., 4.]])
invA = np.linalg.inv(A)
# [[-2.   1. ]
#  [ 1.5 -0.5]]
```

> **实务**：解方程优先 **`solve`**，少直接 `inv(A) @ b`（更慢、数值更差）。

---

### 2）解线性方程组：`np.linalg.solve(A, b)`

- 解 **Ax = b**
- `A` 为 **满秩方阵**；比 `inv(A) @ b` **更稳、更快**

```python
# 2x + y = 5
# x - 3y = -1
A = np.array([[2., 1.], [1., -3.]])
b = np.array([5., -1.])
x = np.linalg.solve(A, b)
# [2. 1.]
```

---

### 3）QR 分解：`Q, R = np.linalg.qr(A)`

- **A = Q R**：Q 列正交，R 上三角
- 用途：最小二乘、正交化、数值稳定回归（概念层）

```python
A = np.array([[1., 2.], [3., 4.], [5., 6.]])
Q, R = np.linalg.qr(A)
```

---

### 4）SVD 奇异值分解：`U, S, VT = np.linalg.svd(A)`

- **A = U Σ Vᵀ**，**任意形状**矩阵均可
- 用途：降维、PCA、伪逆、推荐系统、病态矩阵

```python
U, S, VT = np.linalg.svd(A, full_matrices=False)
# S 为一维奇异值数组
```

---

### 5）其他常用

| API | 作用 |
|-----|------|
| `np.linalg.det(A)` | 行列式（方阵） |
| `np.linalg.matrix_rank(A)` | 矩阵秩 |
| `np.linalg.pinv(A)` | **Moore-Penrose 伪逆**（非方阵 / 奇异） |
| `np.linalg.eig(A)` | 特征值/特征向量（方阵） |
| `np.linalg.norm(x)` | 向量/矩阵范数 |

## 三、一页式速查（必背）

| 主题 | API | 要点 |
|------|-----|------|
| 元素乘 | `A * B` | shape 同，逐元素 |
| 矩阵乘 | `A @ B`、`np.dot(A,B)` | (m,n)@(n,p) |
| 求逆 | `linalg.inv(A)` | 满秩方阵 |
| 解方程 | **`linalg.solve(A,b)`** | 优于 inv@b |
| QR | `linalg.qr(A)` | 正交 × 上三角 |
| SVD | `linalg.svd(A)` | 万能分解 |
| 伪逆 | `linalg.pinv(A)` | 最小二乘/奇异 |

### 易错点

| 易错 | 正确 |
|------|------|
| 用 `*` 做矩阵乘 | 用 **`@`** |
| 用 `inv(A) @ b` 解方程 | 用 **`solve(A, b)`** |
| 奇异矩阵求 `inv` | 用 **`pinv`** 或正则化 |
| 混淆 `@` 与逐元素 `*` | 见上表 |

### 记忆口诀

> **星号逐元素，@ 才是矩阵乘；  
> solve 解 Ax=b，inv 慎用求逆；  
> qr 正交三角，svd 万能分解。**

## 四、演示：OLS 正规方程（概念）

```python
import numpy as np

# y ≈ X @ beta（最小二乘概念；实务用 statsmodels）
X = np.column_stack([np.ones(100), np.random.randn(100)])
beta_true = np.array([1., 2.])
y = X @ beta_true + 0.1 * np.random.randn(100)

# 正规方程 beta = (X'X)^{-1} X'y — 演示用，病态时勿手算
XtX = X.T @ X
Xty = X.T @ y
beta_hat = np.linalg.solve(XtX, Xty)   # 优于 inv(XtX) @ Xty
```

计量推断见第 12 章 **statsmodels**；线代直觉见 [`Econometrics`](../../../Using-Python-for-Introductory-Econometrics/) 册。

---

## 五、【量化专属改造】

1. **因子组合权重**：协方差矩阵 `Σ`，解 `Σ w = μ` 或约束优化 → `solve` / `pinv`。
2. **PCA 降维**：对收益率矩阵 **`svd`** 或 `eig(协方差)`，提取主成分因子。
3. **组合风险**：`var = w @ Sigma @ w`（二次型，矩阵乘 `@`）。
4. **勿手写 inv 回测**：样本协方差接近奇异时用 **`pinv`** 或 ridge 正则。
5. **与 pandas**：`df.values.T @ df.values` 注意对齐；大规模用 **`@`** 配合 `float64`。

---

## 六、与 statsmodels / 全书衔接

- **OLS**：statsmodels 内部 QR/SVD 解最小二乘；理解 `@`、`solve` 有助于读 `summary` 与多重共线性。
- **4.3 ufunc**：`*` 是 `multiply` ufunc；**`@` 不是 ufunc**。
- **4.4**：向量范数、点积 `np.dot(a,b)` 一维时为内积。
- **第 12 章**：`sm.OLS(y, X).fit()` 替代手写线代。
