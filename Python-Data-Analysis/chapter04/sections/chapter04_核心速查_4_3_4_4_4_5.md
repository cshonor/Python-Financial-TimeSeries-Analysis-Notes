## 4.3 ufunc（通用函数）

| 要点 | 内容 |
|------|------|
| 定义 | **逐元素**向量化，C 级速度 |
| 一元 | `sqrt` `exp` `log` `abs` `sign` `ceil` `floor` |
| 二元 | `add` `multiply` `maximum` `minimum` `greater` |
| **`out=`** | 写入已有数组，**省内存** |
| 易错 | `*` 逐元素 vs `@` 矩阵乘 |

---

## 4.4 面向数组编程

| 要点 | API / 规则 |
|------|------------|
| 条件选择 | `np.where(cond, x, y)` |
| 聚合 | `sum/mean/std` + **`axis=0` 按列，`axis=1` 按行** |
| 累计 | `cumsum` `cumprod`（不压维度） |
| 布尔计数 | `(arr > 0).sum()`；`any()` `all()` |
| 排序 | **`arr.sort()` 就地**；**`np.sort(arr)` 副本** |
| 集合 | `unique(..., return_counts=True)`；`isin` / `in1d` |

**口诀**：where 三元，axis 聚合，布尔 sum 计数，sort 就地 np.sort 副本。

---

## 4.5 文件 I/O

| 格式 | API | 用途 |
|------|-----|------|
| `.npy` | `np.save` / `np.load` | **单数组** |
| `.npz` | `savez` / `savez_compressed` | **多数组**（字典 key 读取） |
| 表格 | **`pd.read_csv`** | **实务主力** |

**口诀**：单 npy 多 npz，数组 save 表格 read_csv。

---

## 三节联合易错点

| # | 易错 | 正确 |
|---|------|------|
| 1 | for 循环算元素函数 | **ufunc** |
| 2 | `axis=0` 当按行 | **axis=0 = 按列** |
| 3 | `arr.sort()` 当返回新数组 | 返回 **None**，原数组已变 |
| 4 | 用 npy 存整张行情表 | **read_csv / parquet** |
| 5 | 读 npz 不用 key | `data["name"]` |

## 量化一条龙（4.3→4.4→4.5）

```python
import numpy as np

# 4.3 ufunc：收益
rets = np.diff(np.log(close))

# 4.4：信号 + 净值
signal = np.where(rets > 0, 1, -1)
nav = np.cumprod(1 + rets)

# 4.5：缓存中间结果
np.savez_compressed("cache.npz", rets=rets, nav=nav)
```
