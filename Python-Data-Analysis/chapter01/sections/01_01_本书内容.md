## 一、书本原文核心知识点提炼

### 1.1 本书内容

- **目标**：用 Python 完成数据的操作、处理、清洗、规整；重点是 **语言 + 库 + 工具**，不是全套数据分析方法论。
- **数据范围**：结构化数据（表格/关系型、多维数组、多表关联、时间序列）。
- **定位**：打好「把混乱数据变成规整表格」的能力，为机器学习、数据科学等进阶领域做准备。

原书按 **五类数据分析任务** 组织全书（McKinney 骨架），标准流程一句话：

> **读入 → 清洗整理 → 转换衍生 → 建模分析 → 可视化报告**

---

## 二、数据分析五大类任务（全书骨架 · 可背诵）

### ① 与外部世界交互（读写文件 / 数据库）

**核心**：把数据 **拿进来** 和 **存出去**。

| 类型 | 示例 |
|------|------|
| 文本 | CSV、TSV、TXT |
| 二进制 | Excel、Parquet、HDF5 |
| 数据库 | SQLite、MySQL、PostgreSQL（SQLAlchemy） |
| 网络 | API、JSON、爬虫 |

典型 API：`pd.read_csv()`、`pd.read_excel()`、`pd.to_sql()`

---

### ② 数据准备（清洗、整理、联合、重塑、切片）

**核心**：把原始数据变成 **干净、整齐、可用** 的表（常占项目 **70%–90%** 时间）。

| 动作 | 内容 |
|------|------|
| 清洗 | 去重、缺失值、异常值、格式错误 |
| 整理 | 类型转换、重命名、排序 |
| 联合 | merge、concat、join |
| 重塑 | pivot、stack/unstack、长宽表 |
| 切片 | 行/列筛选、索引 |

典型库：**pandas**（`loc`/`iloc`、`merge`、`pivot`）

---

### ③ 数据转换（统计 / 数学运算 → 新数据集）

**核心**：从已有数据 **计算新字段、新指标**。

| 类型 | 示例 |
|------|------|
| 统计 | 均值、方差、分位数、频数、累计和 |
| 数学 | 对数、差分、标准化、归一化 |
| 特征衍生 | 日期拆分、分箱、哑变量 |
| 分组聚合 | `groupby` + sum/mean/count |

典型库：**NumPy**（向量化）、**pandas**（`groupby`、`apply`）

---

### ④ 建模和计算（统计模型、机器学习）

**核心**：用模型做 **推断、预测、分类**。

| 类型 | 示例 |
|------|------|
| 统计推断 | 假设检验、置信区间、相关/回归 |
| 监督学习 | 回归、分类（逻辑回归、随机森林等） |
| 无监督 | 聚类、降维 |
| 评估 | 训练/测试划分、交叉验证、误差指标 |

典型库：**statsmodels**（推断）、**scikit-learn**（预测/分类）；本书第 12 章侧重 statsmodels + Patsy。

---

### ⑤ 演示（可视化、报告）

**核心**：把结果 **讲清楚、看得见**。

| 类型 | 示例 |
|------|------|
| 可视化 | 折线、柱状、散点、热力、箱线 |
| 工具 | Matplotlib、Seaborn、Plotly |
| 报告 | Jupyter Notebook、PDF、自动化看板 |
