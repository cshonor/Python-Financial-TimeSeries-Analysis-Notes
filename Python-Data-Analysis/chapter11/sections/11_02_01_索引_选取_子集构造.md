#### 11.2.1 索引、选取、子集构造

- 索引可用 `datetime` 或日期字符串（`"2011-01-10"`）。
- **`"2001"`** / **`"2001-05"`** 切片整年/整月。
- 时间切片多为**视图**（修改影响原数据）；也可用 **`truncate(before=, after=)`**。
