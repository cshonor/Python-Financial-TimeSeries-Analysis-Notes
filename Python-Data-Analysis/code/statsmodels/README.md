Statsmodels 是 Python 的统计建模库，侧重计量经济学与统计学模型。

## 主要功能

- 回归分析（OLS、GLS、GLM 等）
- 时间序列（ARIMA、VAR 等）
- 假设检验、ANOVA
- 描述性统计与可视化
