#### 5.2.3 索引、选取、过滤

- **`loc`**：按标签；切片 **含末端**（`loc["b":"c"]`）。
- **`iloc`**：按整数位置；切片规则同 Python（不含 stop）。
- 避免 `[]` 在整数标签上的歧义；**避免链式赋值**，用 `df.loc[mask, "col"] = v`。
