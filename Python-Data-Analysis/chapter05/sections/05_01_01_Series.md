#### 5.1.1 Series

- 一维带标签数组：`values` + `index`；未指定索引则 0…N-1。
- 可由 **dict** 创建；指定 `index` 时按列表对齐，缺键 → `NaN`。
- `pd.isna` / `pd.notna` 检测缺失。
- **算术自动按索引对齐**（类似 join）；Series 与 index 可有 `name`。
