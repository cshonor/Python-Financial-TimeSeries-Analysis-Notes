#### 9.1.3 刻度、标签和图例

- 无参调用如 `ax.xlim()` 为**读取**；传参为**设置**。
- **`set_xticks` / `set_xticklabels`**（`rotation` 防重叠）。
- **`ax.set(title=..., xlabel=..., ylabel=...)`** 批量设置。
- **`ax.legend(loc="best"|"upper left"|...)`**。
