#### 9.2.2 柱状图

- **`plot.bar()`** / **`plot.barh()`**；DataFrame 多列并排；**`stacked=True`** 堆积柱。
- 常与 **`value_counts()`**、**`pd.crosstab()`** 联用。
- **`sns.barplot`** 可带均值与 95% 置信区间误差条。
