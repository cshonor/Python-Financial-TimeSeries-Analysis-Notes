### 7.5 分类数据

- 字典编码：重复小集合 → 整数 codes + categories，省内存、加速 groupby。
- `astype("category")` / `pd.Categorical`；`.cat` 访问器。
- **`pd.get_dummies()`** 独热编码（建模常用）。
