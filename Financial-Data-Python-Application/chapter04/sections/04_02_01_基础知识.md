## 核心定义

`numpy.ndarray` 专为处理 \(n\) 维数组设计，支持将操作整体下推至 C 底层的**矢量化数学计算**。

## 实例化

```python
import numpy as np

np.array([0, 0.5, 1.0, 1.5, 2.0])
np.arange(0, 10)   # 类似 range()，产出 ndarray
```

## 数学运算

- 全局运算：可直接对整个矩阵做标量乘法（`2 * a`）或指数运算（`a ** 2`），彻底规避 `for` 循环
- **ufunc（通用函数）**：`np.exp(a)`、`np.sqrt(a)` 等，底层效率远超纯 Python 的 `math` 模块

## 内置统计归纳

| 方法 | 含义 |
|------|------|
| `a.sum()` | 求和 |
| `a.std()` | 标准差 |
| `a.cumsum()` | 逐项累计求和 |
