## 一、书本原文核心知识点提炼

### 11.1 导入所需的库并加载数据

- **环境**：`pandas` / `numpy` / `matplotlib`；`statsmodels`；评估可用 `sklearn.metrics` 或自写 MAPE。
- **业务**：澳大利亚健康保险委员会月度处方；**测试集 36 个月**，以 **12 个月为预测窗口**滚动，指导产能计划。
- **再现性**：项目开头设 `np.random.seed`（深度学习阶段再加 `tf.random.set_seed`）。

**【个人扩展与补充】**  
<br><br><br><br>

### 11.2 可视化序列及其分量

- **STL**：趋势 + 季节 + 残差。
- **定性**：长期上升；**强年度季节**（年初低、年末高）；**无外生列** → 不需 SARIMAX/VAR。
- **候选**：**SARIMA(p,d,q)(P,D,Q)₁₂**。

**【个人扩展与补充】**  
<br><br><br><br>

### 11.3 对数据进行建模

**平稳化（确立 \(d,D,m\)）**

| 步骤 | ADF \(p\) | 结论 |
|------|-----------|------|
| 原始 | 1.0 | 不平稳 |
| 一阶差分 \(d=1\) | 0.12 | 仍不平稳 |
| 再季节差分 \(D=1, m=12\) | 0.0 | 平稳 |
| **锁定** | | **SARIMA(p,1,q)(P,1,Q)₁₂** |

**11.3.1 模型选择（网格 + AIC）**

- 训练 / 测试：除最后 **36** 月外为训练集。
- \(p,q,P,Q \in [0,4]\) 共 **625** 组；AIC 最小：**SARIMA(2,1,3)(1,1,3)₁₂**。

**11.3.2 残差分析**

- 四宫格：无趋势、近正态、Q-Q 贴线、ACF 无显著滞后。
- **Ljung-Box**：各滞后 \(p>0.05\) → 白噪声 → 可预测。

**【个人扩展与补充】**  
<br><br><br><br>

### 11.4 预测和评估模型的性能

- **滚动**：36 月测试集上 **3 次 × 12 月** 块预测，每块后用真实值扩展再训。
- **基线**：**季节朴素**——用过去 12 月作为未来 12 月预测。
- **MAPE**：朴素 **12.69%**；SARIMA **7.90%** → 显著优于基线，可投产。

**【个人扩展与补充】**  
<br><br><br><br>

### 11.5 下一步

- **统计学闭环**：基线对比 + 残差白噪声检验，防过拟合。
- **瓶颈**：\(>10{,}000\) 点、高维、强非线性 → 统计模型效率与表达能力不足。

## 二、关键语法速查表

| 场景 | API |
|------|-----|
| STL | `STL(y, period=12, robust=True).fit()` |
| 季节差分 | `y.diff(12)` 或 `y.diff().diff(12)` |
| SARIMA | `SARIMAX(y, order=(p,d,q), seasonal_order=(P,D,Q,m))` |
| 网格 | `itertools.product` + `res.aic` |
| 残差 | `plot_diagnostics` / `acorr_ljungbox` |
| 块滚动 | 每 12 步 `fit` → `forecast(12)` → 拼接真实测试段 |
