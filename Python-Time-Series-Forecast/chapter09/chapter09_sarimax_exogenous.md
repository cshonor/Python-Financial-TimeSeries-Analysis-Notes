## 一、书本原文核心知识点提炼

### 9.1 研究 SARIMAX 模型

- **内生变量**：要预测的 \(y_t\)。**外生变量**：外部预测因子 \(X_t^i\)。
- **形式**：\(y_t = \text{SARIMA}(p,d,q)(P,D,Q)_m + \sum_i \beta_i X_t^i\)（线性；分类需编码）。
- **统一 API**：`SARIMAX` — 无季节 → ARIMAX；无外生 → SARIMA；都无 → ARIMA。

#### 9.1.1 宏观经济数据与 p 值纠偏

- 数据：1959—2009 季度宏观序列；目标 **realgdp**（实际 GDP）。
- **误区**：`summary` 里外生变量 **p>0.05** 就删掉。
- **正解**：p 值只检验「系数是否显著异于 0」，**不能**判断预测是否有用。
- **选模**：保留候选外生变量，用 **AIC 最小** 防过拟合，勿按 p 值做特征筛选。

#### 9.1.2 使用 SARIMAX 的注意事项

- 预测 \(y_{t+h}\) 需要 **同期外生变量** \(X_{t+h}\)。
- **多步**：须先**预测外生变量** → 外生预测误差传导 → **目标误差放大**。
- **实践**：优先 **单步**（`window=1`）滚动；每步等到真实 \(X\) 再预测下一步。外生极易知（确定性日历）时才考虑多步。

### 9.2 使用 SARIMAX 预测实际 GDP

- **ADF**：GDP 不平稳；一阶差分 \(p\approx 3.3\times 10^{-8}\) → **\(d=1, D=0\)**（无显式季节）。
- **外生**：消费、私人投资等 5 变量；网格 → **SARIMAX(3,1,3)(0,0,0)\_4** ≈ **ARIMAX(3,1,3)**。
- **残差**：Q-Q、Ljung-Box 全 \(p>0.05\)。
- **滚动**：`WINDOW=1`；差分预测 **cumsum 逆变换**。
- **MAPE**：SARIMAX **0.70%** vs 末值基线 **0.74%**（宏观尺度下 0.04% 亦重大）。

### 9.3 下一步

- SARIMAX 假设 **单向**：\(X\to y\)，非 \(y\to X\)。
- 双向、多变量 → 第 10 章 **VAR**。

### 9.4 练习与小结

1. SARIMAX 可引入外生变量。  
2. **平稳化通常只对目标 \(y\)**，不对 \(X\)（书中原则；实践中常对 \(X\) 做对齐/缩放，但不随意差分外生）。  
3. **多步需预测 \(X\)** → 误差累积；推荐 **单步**。

## 二、关键语法速查表

| 场景 | API |
|------|-----|
| 拟合 | `SARIMAX(endog, exog=exog, order=..., seasonal_order=...).fit()` |
| 预测 | `res.forecast(steps=1, exog=exog_future)` |
| 滚动 | 每步 `fit` + `forecast(1)` + 观测新 \(X\) |
| 选模 | 固定 \(d,D,m\)，AIC 最小；**勿按 p 删 exog** |

## 四、【量化专属改造】金融 SARIMAX

| 外生变量 | 示例 |
|----------|------|
| 宏观 | 利率、CPI、PMI、社融 |
| 市场 | 指数收益、VIX、汇率 |
| 事件 | 节假日哑变量、财报发布日 |
| 单步 | 日频因子：用 **T 日收盘因子预测 T+1 收益**（外生已知） |
| 多步 | 须模拟宏观路径或只用 SARIMA |

**量化警示**：用「预测出的因子」再预测收益 → 双重误差；回测中用 **已实现因子** 与实盘一致。
