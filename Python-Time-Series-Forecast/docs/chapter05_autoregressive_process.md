## 一、书本原文核心知识点提炼

### 5.1 预测零售店平均每周客流量

- **场景**：约 1000 周数据（自 2000 年）；预测下周客流以优化排班。
- **特征**：长期趋势 → **非平稳**；无明显季节（本例）。
- **差分后 ACF**：**指数缓慢衰减**或正弦式波动 → **非 MA、非 RW** → **AR**，需 **PACF** 定阶。

### 5.2 定义自回归过程

- **AR(p)**：\(y_t = C + \phi_1 y_{t-1} + \cdots + \phi_p y_{t-p} + \epsilon_t\)。
- \(\phi_i\)：滞后 \(i\) 对当前的线性影响。
- **随机游走**：AR(1) 且 \(\phi_1=1\) → \(y_t = C + y_{t-1} + \epsilon_t\)（带/不带漂移）。
- **阶数 \(p\)**：依赖过去 \(p\) 个观测；拟合参数个数随 \(p\) 增。

### 5.3 求平稳自回归过程的阶数

- **ACF 局限**：\(y_t\) 与 \(y_{t-k}\) 含**间接路径**（经中间滞后传递）。
- **PACF**：控制中间滞后后，\(y_t\) 与 \(y_{t-k}\) 的**净相关**。
- **定阶法则**：平稳 AR(p) 的 PACF 在 **lag \(p\) 之前显著**，**\(p+1\) 起落入置信带**。
- **书中案例**：差分后客流 → **AR(3)**（PACF 在 lag 3 后截断）。

### 5.4 预测自回归过程

- **划分**：训练 947 周，测试最后 **52 周**。
- **基线**：[第 2 章](./chapter02_baseline_forecasting.md) 历史均值、末值；AR(3) 须更优。
- **实现**：`rolling_forecast`，`window=1`；**`SARIMAX(..., order=(3, 0, 0))`**。
- **书中 MSE（差分尺度）**：AR(3) **0.92** < 末值 **1.45** < 历史均值 **3.11**。
- **逆变换**：`cumsum()` + 训练末水平值；**MAE ≈ 3.45** 人/周（原尺度业务解释）。

### 5.5 下一步

- ACF 慢衰减 + PACF 在 \(p\) 截断 → **AR(p)**。
- **ACF 与 PACF 都慢衰减** → **[ARMA](./chapter06_arma_modeling.md)**，非纯 AR 或纯 MA。

### 5.6 练习与本章小结

- `ArmaProcess` 模拟 AR(2)；端到端：ADF → 差分 → ACF/PACF → 滚动预测 → MSE。

**小结**：

1. AR：当前值与过去值、误差线性相关。  
2. ACF 缓慢衰减 → 考虑 AR。  
3. PACF 测「纯净」滞后相关。  
4. PACF 在 \(p\) 后截断 → 阶数为 \(p\)。

## 四、【量化专属改造】金融 AR 建模

| 要点 | 实践 |
|------|------|
| 序列选择 | 价格常非平稳 → 对 **收益** 或 **对数收益** 拟合 AR；水平价先差分 |
| 频率 | 日频 AR(p) 的 \(p\) 通常小；周/月频可略大 |
| PACF 定阶 | 与 AIC/BIC、`ar_select_order` 交叉验证 |
| 与 MA 区分 | ACF 截断 → MA（第 4 章）；ACF 拖尾 + PACF 截断 → AR |
| 滚动回测 | 扩展窗口/滑动窗口重估 \(\phi\)，避免一次拟合用全样本 |
