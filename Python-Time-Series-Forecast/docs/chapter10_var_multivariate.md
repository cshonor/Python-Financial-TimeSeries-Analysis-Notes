## 一、书本原文核心知识点提炼

### 10.1 研究 VAR 模型

- **突破单向限制**：VAR 允许捕捉多个序列随时间的相互作用；每个序列的过去值都会影响其他序列的现在，**反之亦然**。
- **VAR(p) 与 AR(p)**：VAR 是第 5 章 AR 的推广。AR 中 \(y_t\) 仅依赖自身滞后；VAR 中 \(y_{1,t}\) 还依赖 \(y_{2,t-1}\) 等**跨序列滞后**。
- **阶数 \(p\)**：有多少期滞后（含自身与他序列）进入当前方程。
- **前提**：与 AR 相同，**所有序列必须平稳**（建模前差分等）。
- **因果检验动机**：形状相似不足以证明「A 能预测 B」，需 **Granger 检验**。

### 10.2 设计 VAR(p) 建模过程（含 Granger）

流程与 ARMA 类似（平稳化 → AIC 定阶 → 残差分析），但在 **AIC 定阶与残差分析之间** 必须插入 **Granger 因果检验**。

1. **平稳性 + AIC 定阶**：变换平稳后，遍历候选 \(p\)，取 AIC 最小的 VAR(p)。  
2. **Granger 因果（阻断步骤）**  
   - 定义：\(y_1\) 的过去是否在预测 \(y_2\) 时具有统计显著性；若是，称 \(y_1\) 为 \(y_2\) 的 **Granger 因**。  
   - \(H_0\)：\(y_1\) **不是** \(y_2\) 的 Granger 因。  
   - **\(p<0.05\)**：拒绝 \(H_0\)。  
   - **双向**：须分别检验 \(y_1\to y_2\) 与 \(y_2\to y_1\)，**两者均通过** 才适合 VAR。  
   - **失败**：\(p\ge 0.05\) → VAR 失效，退回 **SARIMAX** 逐序列建模。  
3. **残差分析**：Q-Q、Ljung-Box；\(p>0.05\) 表明白噪声，方可预测。

### 10.3 预测实际可支配收入和实际消费

- **ADF**：原始 `realdpi` / `realcons` \(p=1.0\)（不平稳）；一阶差分后 \(p<0.05\)。  
- **划分**：80% 训练 / 20% 测试；\(p\in[0,14]\) 网格，**VAR(3)** AIC 最小。  
- **Granger**：`realcons → realdpi`、`realdpi → realcons` 均 \(p<0.05\) → 双向成立。  
- **残差**：两方程 Ljung-Box 滞后 \(p>0.05\)。  
- **滚动预测**：差分上 **4 步**滚动（一季一步 × 4 季），输出后 **`cumsum` 逆变换**。  
- **MAPE**：`realcons` VAR **优于** 基线；`realdpi` VAR **3.14%** vs 基线 **2.26%** → 统计上通过 Granger，样本外仍可能不如单序列 SARIMAX。

### 10.4 下一步

- **统计学篇收官**：MA → AR → ARMA → ARIMA → SARIMA → SARIMAX → **VAR**。  
- **瓶颈**：点数 **>1 万**、高维特征时，纯统计模型效率低、难拟合复杂关系。  
- **转折**：第 11 章顶点项目后进入 **深度学习** 时间序列预测。

### 10.5 练习与本章小结

- **10.5.1**：`statsmodels` 用 **VARMAX** 实现 VAR → 可扩展 **VARMA(p,q)**。  
- **10.5.2**：带外生变量的 **VARMAX(p,q)**；重走流程，看能否击败基线。  
- **小结**：  
  1. VAR(p) 捕获多序列**相互作用与双向关系**。  
  2. 应用铁律：双向 **Granger** 均通过。  
  3. \(H_0\)：A 不是 B 的 Granger 因；**\(p<0.05\)** 拒绝 \(H_0\)。

## 四、【量化专属改造】金融 VAR

| 场景 | 做法 |
|------|------|
| 股债商 | 收益向量 VAR；宏观冲击传导 |
| 行业指数 | 板块收益联立 |
| 平稳 | 用收益率，非价格水平 |
| Granger | 检验「因子→收益」不等价因果；谨慎解读 |
| 高维 | 变量过多 → 降维、LASSO-VAR 或深度学习 |
