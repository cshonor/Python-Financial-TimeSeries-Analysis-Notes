## 一、书本原文核心知识点提炼

### 2.1 定义基线模型

- **基线**：最简单方案，用统计量或规则出预测，**不拟合参数**；用于判断复杂模型是否「真的更好」。
- **样本外评估**：按时间切分（如 1960—1979 训练，**1980 四季**测试），预测未来再与实测对比。

### 2.2 预测历史均值

- 用训练集**全体**算术平均 \(\bar{y}\)，未来各期预测均为 \(\bar{y}\)。
- 书中均值约 **4.31** 美元；1980 四季预测同值。
- **MAPE ≈ 70%**：序列有**强趋势**，早期低值拉低均值，无法代表近期水平。

### 2.3 预测最后一年的均值

- 仅用训练集**最后一年**（1979 四季）均值，约 **12.96** 美元，作为 1980 预测。
- **MAPE ≈ 15.60%**：未来更依赖**近期**而非遥远过去 → **自相关**迹象。

### 2.4 使用最后已知数值

- 预测值 = 训练集**最后一个观测**（1979 Q4 ≈ **9.99**），各期相同。
- **MAPE ≈ 30.45%**：差于「去年均值」；因 **Q4 季节性低谷**，复制末值忽略 Q1—Q3 较高模式。

### 2.5 简单季节性预测（Naive Seasonal）

- 取上一**完整周期**（1979 四季）逐季复制到 1980 Q1—Q4。
- **MAPE ≈ 11.56%**（本章最佳）：说明 **季节性** 主导，复杂模型须纳入季节分量。

### 2.6 下一步（小结）

1. 预测项目**必须先有基线**。  
2. 基线 = 均值 / 上周期 / 末值等启发式。  
3. **MAPE** 越小越好：\(\text{MAPE} = \frac{100}{n}\sum\left|\frac{y-\hat{y}}{y}\right|\)。  
4. 复杂模型 MAPE 须 **< 11.56%** 才有价值；**随机游走** 例外见第 3 章。

## 二、基线方法速查表

| 方法 | 预测规则 | 适用 |
|------|----------|------|
| 历史均值 | \(\hat{y}_{t+h}=\bar{y}_{\text{train}}\) | 平稳、无趋势季节 |
| 去年均值 | \(\hat{y}=\text{mean}(\text{last year})\) | 有趋势、弱季节 |
| 末值 | \(\hat{y}=y_T\) | 随机游走类（第 3 章） |
| 季节朴素 | 重复最近 \(m\) 期（季频 \(m=4\)） | 强季节 |

## 四、【量化专属改造】金融基线

| 书本基线 | 量化用法 |
|----------|----------|
| 历史均值 | 全样本收益均值 → 零预测基准 |
| 去年/滚动均值 | 滚动 20 日均价、过去一年波动率 |
| 末值 | 日频收盘价单步预测基线（第 3 章 RW） |
| 季节朴素 | 重复上周同日、上月同日模式；农历/财报季 |
| MAPE | 也可用 MAE、RMSE；价格序列常对**收益**算误差 |
| 时间切分 | 滚动 walk-forward，[`chapter01`](../code/chapter01_forecast_workflow_demo.py) |

**门槛**：因子/模型样本外 IC、夏普须显著优于「季节朴素收益」或「零预测」。

## 五、与 statsmodels / 后续章节衔接

- 基线不需 statsmodels；评估逻辑与第 1 章 **train/test 按时间切分** 一致。
- **MAPE 优于 11.56%** 后，才值得上 ARMA/SARIMA（季节阶数 \(m=4\)）。
- 第 3 章：若序列是随机游走，**末值基线**已是理论最优单步，MAPE 可能仍「好看」但无结构可学。
