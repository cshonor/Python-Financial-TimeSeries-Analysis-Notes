## 一、书本原文核心知识点提炼

### 18.1 了解顶点项目

- **业务**：电网负荷规划，避免过发/欠发。  
- **数据**：UCI 家庭用电，2006-12～2010-11，**每分钟**，200 万+ 行，9 列。  
- **目标**：预测未来 24h **`Global_active_power`**（有功功率）。

**【个人扩展与补充】**  
<br><br><br><br>

### 18.2 数据整理和预处理

**18.2.1 缺失值**  
- `Sub_metering_3` 约 1.25% 缺失；最长连续约 5 天空白 → **整列删除**（避免伪趋势）。

**18.2.2 类型**  
- 传感器读数 `object` → **`float64`**。

**18.2.3 重采样**  
- 分钟级噪声大、无业务意义 → **`resample('h').sum()`** → 约 **34,949** 行，适合 DL。

**【个人扩展与补充】**  
<br><br><br><br>

### 18.3 特征工程

**18.3.1 删列**  
- `Sub_metering_1` 约 75% 为 0、方差极小 → 删除。

**18.3.2 傅里叶与周期**  
- FFT：周季节不明显；**日季节峰值明显** → sin/cos **日周期**编码。

**18.3.3 拆分与缩放**  
- **70 / 20 / 10** 时序划分；`MinMaxScaler` **仅 fit 训练集**。

**【个人扩展与补充】**  
<br><br><br><br>

### 18.4 建模准备

- `seed(42)`（TF + NumPy）可再现。  
- 复用 **`DataWindow`**：输入 24h → 标签 24h。  
- **`compile_and_fit`**：MSE + Adam + MAE + `EarlyStopping(patience=3)`。

**【个人扩展与补充】**  
<br><br><br><br>

### 18.5 七大模型（多步 24h 有功功率）

| 编号 | 模型 | 要点 |
|------|------|------|
| 18.5.1 | 基线 | 重复末值 / 重复过去 24h 曲线 |
| 18.5.2 | 线性 | 无隐藏层 `Dense` |
| 18.5.3 | DNN | 2×64 ReLU |
| 18.5.4 | LSTM | 32 units, `return_sequences=True` |
| 18.5.5 | CNN | 输入长 **26**（\(24+3-1\)），`Conv1D` |
| 18.5.6 | CNN+LSTM | Conv1D → LSTM |
| 18.5.7 | ARLSTM | `LSTMCell` 逐步反馈 |
| 18.5.8 | **冠军** | 测试集 **ARLSTM MAE = 0.074** 最低 |
