## 一、书本原文核心知识点提炼

### 6.1 预测数据中心带宽使用量

- **任务**：预测每小时带宽（Mbit/s），优化算力与维护。
- **数据**：约 10000 点；**有趋势、无显式季节** → 非平稳。
- **差分后**：ACF、PACF **均显著且拖尾** → **ARMA**，非纯 AR/MA。

### 6.2 研究自回归移动平均过程

- **ARMA(p,q)**：\(y_t = C + \sum_{i=1}^p \phi_i y_{t-i} + \epsilon_t + \sum_{j=1}^q \theta_j \epsilon_{t-j}\)（书中常合并常数项写法）。
- **特例**：\(p=0\) → MA(q)；\(q=0\) → AR(p)。

### 6.3 确定平稳 ARMA 过程

- ARMA 的 ACF/PACF **无清晰截断**，图形法**不能**可靠定 \(p,q\)。
- 模拟真 ARMA(1,1) 仍可在 lag>1 见显著 → 必须用**数值流程**。

### 6.4 通用建模过程（核心）

**第 1 步**：ADF → 不平稳则差分至平稳。

**第 2 步：网格搜索 + AIC（6.4.1～6.4.2）**

- \(p,q \in \{0,1,2,3\}\) 等组合逐一 `SARIMAX(p,0,q)` 拟合。
- **AIC** 越小越好（惩罚参数个数，抑制过拟合）。
- 选 **AIC 最小** 为候选。

**第 3 步：残差分析（6.4.3～6.4.4）**

- 残差 = 实际 − 拟合；应像**白噪声**（随机、不相关、近似正态、均值 0）。
- **Q-Q 图**：点贴近直线 → 正态。
- **残差 ACF**：除 lag0 外不显著。
- **Ljung-Box**：\(H_0\)=残差独立；**希望 \(p>0.05\)**（不能拒绝 \(H_0\) → 视为无自相关）。\(p<0.05\) 则改 \(p,q\)。

### 6.5 应用通用流程（带宽案例）

- 水平 ADF \(p≈0.80\) → 一阶差分 ADF \(p≈0\)。
- 测试集最后 **168 点**（7 天）。
- 16 组 \((p,q)\) → **ARMA(2,2)** AIC 最小（27991）。
- 残差：Q-Q、ACF、Ljung-Box 均通过。

### 6.6 预测带宽

- **滚动预测**：含 MA(2) 时单步 horizon ≤2；`window=2` 滚过测试集。
- **逆变换**：`cumsum` + 差分前末值。
- **MSE（差分）**：ARMA(2,2) **1.8** < 末值 **2.2**；**MAE（原尺度）≈14 Mbit/s**。

### 6.7～6.8 小结

1. ARMA 双图拖尾 → 不用截断定阶。  
2. **AIC 选相对最优**（越小越好）。  
3. **残差分析验绝对质量**。  
4. 白噪声：Q-Q 直线 + **Ljung-Box \(p>0.05\)**。  
5. 下一章 **ARIMA** 加入 \(d\)。

## 二、通用工作流速查

```text
ADF → (差分) → 网格 (p,q) → AIC 最小 → 残差诊断 → 滚动预测 → cumsum 还原
```

| 步骤 | API / 工具 |
|------|------------|
| 平稳 | `adfuller`, `y.diff()` |
| 拟合 | `SARIMAX(y, order=(p,0,q)).fit().aic` |
| 残差 | `res.resid`, `acf(resid)`, `ljungbox` |
| 预测 | `rolling_forecast`, `steps≤max(p,q)` |
| 还原 | `y0 + fc.cumsum()` |

## 四、【量化专属改造】金融 ARMA

| 要点 | 实践 |
|------|------|
| 序列 | 对**收益率**拟合 ARMA 更常见；价格水平用 ARIMA(,,) 或先差分 |
| 网格范围 | \(p,q\) 不宜过大；可用 `pmdarima.auto_arima` 自动化（扩展） |
| 残差检验 | LB \(p<0.05\) 提示未建模结构（如 GARCH、制度切换） |
| 滚动 | 样本外逐期重估或扩展窗口，与回测一致 |
| 与第 2 章 | 样本外 MSE 须击败基线 |
