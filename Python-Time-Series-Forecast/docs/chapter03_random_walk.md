## 一、书本原文核心知识点提炼

### 3.1 随机游走过程

- **定义**：每一步上升/下降概率相等；金融价格常见，长趋势与突然反转并存。
- **模型**：\(y_t = C + y_{t-1} + \epsilon_t\)，\(\epsilon_t\) 为白噪声（常设均值 0、方差 1）。
- **漂移（Drift）**：\(C \neq 0\) → 带漂移随机游走。
- **模拟**：对白噪声**累加**（cumsum）即得随机游走。

### 3.2 识别随机游走

**正式定义**：**一阶差分平稳且不相关**的序列（差分后为白噪声）。

#### 3.2.1 平稳性与差分

- **平稳**：均值、方差、自相关不随时间变；MA/AR/ARIMA 等要求（或经变换后）平稳。
- **差分**：相邻差 \(y_t - y_{t-1}\)，去趋势、稳定均值；**对数**常用来稳定方差。

#### 3.2.2 ADF 检验

- **Augmented Dickey-Fuller**：\(H_0\) = 存在单位根（**不平稳**）。
- **\(p < 0.05\)**：拒绝 \(H_0\)，认为平稳。

#### 3.2.3 自相关函数（ACF）

- 度量滞后 \(k\) 与当前值的线性相关。
- 随机游走 **一阶差分后**：除 lag 0 外**无显著自相关**（近似白噪声）。

#### 3.2.4～3.2.5 GOOGL 验证（书中流程）

1. 收盘价 ADF：\(p > 0.05\) → 不平稳。  
2. 一阶差分后再 ADF：\(p < 0.05\) → 平稳。  
3. 差分序列 ACF 无显著结构 → **近似随机游走**。

### 3.3 预测随机游走

- **不能**用统计学习/深度学习「从随机性中学习」；用第 2 章**简单基线**。

#### 3.3.1 长期多步预测失效

- 多步（如 200 步）时，随机项不断累加，**方差爆炸**；均值/漂移/末值法长期皆差。

#### 3.3.2 单步短期预测

- **合理场景**：只预测 **\(t+1\)** 一步。
- **最佳策略**：\(\hat{y}_{t+1} = y_t\)（最后已知值）；MSE 可很小但是「平移」而非学到结构。
- **实践**：多次滚动单步预测，避免误差累积。

### 3.4 下一步

- 随机游走 = 预测边界；若差分平稳且 ACF **有**显著滞后 → 可建 **MA / AR / ARMA**（MA 见 [第 4 章](./chapter04_moving_average_process.md)）。

## 二、关键语法速查表

| 场景 | API |
|------|-----|
| 模拟 RW | `np.cumsum(rng.normal(0, 1, n))` |
| 一阶差分 | `y.diff()` 或 `y.diff(1).dropna()` |
| ADF | `statsmodels.tsa.stattools.adfuller(y)` |
| ACF 图 | `statsmodels.graphics.tsaplots.plot_acf(diff)` |
| 单步预测 | `y.shift(1)` 作预测；或 `forecast = y.iloc[-1]` |
| MSE | `((actual - pred)**2).mean()` |

## 四、【量化专属改造】金融随机游走含义

| 概念 | 量化解读 |
|------|----------|
| 价格水平 RW | 对数价格近似 RW → **收益近似白噪声**；有效市场常见表述 |
| 差分 = 收益 | `close.pct_change()` 与 `close.diff()` 分析对象 |
| ADF on 价格 | A 股日收盘常不平稳；在**收益**上做 ADF/ACF |
| 单步预测 | 日频「明日收盘 ≈ 今日收盘」仅作**基线**；策略需超越该 MSE |
| 长期预测 | 期权/风险用模拟路径；勿用单模型点预测 200 日价格 |
| 与第 1 章 | 禁止 shuffle；滚动单步评估：[`05_shift_and_signal_alignment.md`](../code/time_series_quant/05_shift_and_signal_alignment.md) |
