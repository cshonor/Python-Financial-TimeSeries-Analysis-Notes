## 一、书本原文核心知识点提炼

### 15.1 探索递归神经网络 (RNN)

- **RNN**：适合**顺序重要**的序列；LSTM、GRU 为其子类。  
- **隐藏状态 \(h_t\)**：当前输入 \(x_t\) 与上一时刻 \(h_{t-1}\) 共同决定 \(h_t\)，实现**反馈记忆**。  
- **缺陷**：**短期记忆**；长序列上早期梯度→0（**梯度消失**），难以学习远期依赖。

**【个人扩展与补充】**

### 15.2 研究 LSTM 架构

- **单元状态 \(C_t\)**：长期信息通道，缓解梯度消失。  
- **15.2.1 遗忘门**：sigmoid\((h_{t-1}, x_t)\) → 与 \(C_{t-1}\) **逐点乘**，决定丢弃什么。  
- **15.2.2 输入门**：sigmoid 选通道 × tanh 候选 → **加**到 \(C_t\)。  
- **15.2.3 输出门**：sigmoid × tanh\((C_t)\) → 新 \(h_t\)，传向下一步或输出层。

**【个人扩展与补充】**  
<br><br><br><br>

### 15.3 实现 LSTM 架构

- **通用层**：`LSTM(32, return_sequences=True)` — 保留**逐步**隐藏状态供后续使用。  
- **损失/优化**：与第 14 章相同 — **MSE** + **Adam** + **MAE** + EarlyStopping。

**15.3.1 单步**

- 输入 **过去 24h**，预测 **下一时刻流量**。  
- `LSTM(32, return_sequences=True)` → 取末步 → `Dense(1)`。  
- **MAE**：优于基线、线性、DNN。

**15.3.2 多步**

- 输入 24h → 输出 **未来 24h**（`shift=24`）。  
- 同结构；`Dense(1)` 逐步输出 → `Reshape(24)`。  
- 测试 **MAE ≈ 0.058**（DNN **0.064**，线性 **0.076**）。

**15.3.3 多输出**

- 输入 24h → 下一步 **流量 + 温度**。  
- `Dense(2)`；**MAE ≈ 0.017**（多输出 DNN **0.020**）。

|------|-------|
