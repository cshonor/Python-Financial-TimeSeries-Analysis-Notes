## 一、书本原文核心知识点提炼

### 7.1 定义差分自回归移动平均模型

- **ARIMA(p,d,q)** = 对**差分 \(d\) 次后平稳**的序列做 ARMA(p,q)。
- 差分序列 \(y'_t\) 满足 ARMA 形式（常数、\(\phi_i y'_{t-i}\)、\(\theta_j \epsilon_{t-j}\)、白噪声 \(\epsilon_t\)）。
- **\(d\)**：使序列平稳的**最小差分次数**（积分 = 差分的逆）。
- **\(d=0\)**：ARIMA(p,0,q) ≡ ARMA(p,q)。

### 7.2 修改通用建模过程

1. **ADF**：若不平稳，尝试差分 1 次、2 次… 记录最小 **\(d\)**（实践中 \(d\le 2\) 常见）。
2. **固定 \(d\)** 后，对 **\(p,q\)** 网格拟合 ARIMA(p,d,q)，**AIC 最小** 选模。  
   - **⚠️ 比较 AIC 时 \(d\) 必须相同**；否则似然不可比。
3. **残差分析**：Q-Q、残差 ACF、**Ljung-Box \(p>0.05\)**（同第 6 章）。

### 7.3 预测非平稳时间序列（J&J 季度 EPS）

- **ADF**：水平 \(p=1.0\)；一阶差分 \(p=0.9\)；**二阶差分 \(p=0.006\)** → **\(d=2\)**。
- **测试集**：最后一年（1980，4 个季度）。
- **网格**：\(p,q\in\{0..3\}\)，\(d=2\) 固定 → **ARIMA(3,2,3)** AIC 最小。
- **残差**：定性白噪声 + Ljung-Box 全 \(p>0.05\)。
- **MAPE**：季节朴素（1979 四季复制）**11.56%**；ARIMA(3,2,3) **2.19%**。

### 7.4 下一步 & 7.5 练习

- **优势**：原尺度拟合与预测，省去 ARMA 手工逆变换。
- **局限**：强生数据有**季度季节**；ARIMA 不建模季节 → [第 8 章 SARIMA](./chapter08_sarima_seasonality.md)。
- **练习**：用 ARIMA 重跑第 4～6 章数据集。

**小结**：

1. ARIMA = AR + I(d) + MA。  
2. 直接用于非平稳序列，原尺度预测。  
3. \(d\) = 最小平稳化差分次数。  
4. \(d=0\) ≡ ARMA。

|------|-----|

## 四、【量化专属改造】金融 ARIMA

| 要点 | 实践 |
|------|------|
| 价格序列 | 常 \(d=1\)（对数价格或一阶差分收益）；\(d=2\) 较少 |
| 与 ARMA 流程 | 仍建议样本外滚动；ARIMA 减少逆变换代码 |
| 季节 | 有季报/月报周期 → 优先 SARIMA（第 8 章） |
| AIC | 同一 \(d\) 下比 \((p,q)\)；勿跨 \(d\) 比 AIC |
