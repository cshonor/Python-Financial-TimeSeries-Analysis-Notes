# Python-Financial-BigData-Analysis

对应教材：**《Python 数据金融大数据分析》**。

本册侧重：**金融表格数据预处理、多标的结构、因子与回测数据准备**。

## 目录说明

| 子目录 | 用途 |
|--------|------|
| `docs/` | 全书章节笔记（五段式模板，逐步补齐） |
| `code/pandas/` | **当前主力**：Pandas 量化笔记与示例（原仓库 `Pandas/`） |
| `dataset/` | 行情、因子等离线数据 |
| `exercise/quant_practice_cases/` | 量化综合练习与 NumPy+Pandas 入门脚本 |
| `assets/` | 图表与导图 |

## `code/pandas/` 模块导航

| 目录 | 主题 |
|------|------|
| `00_core_objects/` | Series / DataFrame、金融存储约定 |
| `01_core_data_operations/` | 读取、筛选、聚合、合并（进阶） |
| `02_data_cleaning_preprocessing/` | 缺失、离群、重复、复权 |
| `04_multi_asset_data_handling/` | MultiIndex、长宽表、分组回测 |
