在进行复杂的切片或广播运算前，需要通过对象自有属性审查当前变量内部形态，保障代码的稳健性。

## 关键特征属性

| 属性 | 含义 | 示例 |
|------|------|------|
| `g.ndim` | 数组维度数 | `2` |
| `g.shape` | 各维度元素跨度 | `(3, 4)` |
| `g.size` | 总元素个数 | `12` |
| `g.itemsize` | 单元素字节占用 | `8`（float64） |
| `g.dtype` | 数据类型 | `float64`, `int32` |

```python
g = np.zeros((3, 4))
g.ndim, g.shape, g.size, g.dtype
```
