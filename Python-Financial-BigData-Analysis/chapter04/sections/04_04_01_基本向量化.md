## 关键定义

放弃处理微观粒度的单一数字操作，升维至一次性对一整套数组集落指令。这是提升速度与可读性的灵魂思路。

## 广播机制（Broadcasting）

使不同形状的数组能够直接实施四则运算的幕后推手。

**规则示例**：体积为 `(4, 3)` 的矩阵与长度为 3 的一维向量运算时，NumPy 自动将向量**按行广播**，使形状对齐后逐元素运算。

```python
a = np.ones((4, 3))
b = np.array([1, 2, 3])
a + b   # (4, 3) + (3,) → 广播成功
```

## 易错细节：广播断裂

将 `(4, 3)` 结构与长度为 4 的向量强行匹配时，尾部尺寸 3 与 4 不可调和：

```
ValueError: operands could not be broadcast together with shapes (4,3) (4,)
```

必须显式通过 `reshape` 等方式调整布局后再运算。
