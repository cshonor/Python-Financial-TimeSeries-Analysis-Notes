## 矩阵初始化捷径

| API | 用途 |
|-----|------|
| `np.zeros()` / `np.ones()` | 快速填充 0 或 1 矩阵 |
| `np.empty()` | 不覆写清理的内存分配，速度最快，但会残留随机脏数据 |
| `np.eye()` | 主对角线全为 1 的方阵（线代常用） |
| `np.linspace(start, end, num)` | 等差数组生成器 |

## 轴向计算（Axis 核心机制）

以 2D 矩阵 `b` 为例：

- `b.sum(axis=0)`：沿第 1 个轴**垂直向下按列**求和
- `b.sum(axis=1)`：沿第 2 个轴**水平向右按行**求和

```python
b = np.array([[1, 2, 3], [4, 5, 6]])
b.sum(axis=0)  # [5, 7, 9]  列和
b.sum(axis=1)  # [6, 15]   行和
```
