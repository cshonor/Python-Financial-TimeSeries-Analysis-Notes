## 关键定义

Python 内置提供了专用的 `array` 模块。与可容纳异构数据的标准列表不同，它以一种**极其紧凑的形式**表示基本数据类型（如字符、整数、浮点数）的数组。

## 实用要点

- 在内存管理上有优势
- 支持 `.tofile()` 和 `.fromfile()`，以底层二进制形式进行大文件的磁盘读写

```python
import array

a = array.array('f', [0.5, 0.75, 1.0, 1.5, 2.0])
```

## 局限性与易错点

| 问题 | 说明 |
|------|------|
| **单向类型约束** | 实例化时必须指定类型代码（如 `'f'` 表示浮点），元素类型必须绝对统一 |
| **类型崩溃** | 向浮点 `array` 追加字符串会抛出 `TypeError: must be real number, not str` |
| **缺乏高维拓扑** | 严格限制为**一维数组**，无法直接表示相关性矩阵、协方差矩阵等多维结构 |
