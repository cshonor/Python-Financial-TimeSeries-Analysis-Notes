# 4.1.1 用 Python 列表形成数组

> 所属：[4.1 数据数组](./04_01_数据数组.md) · [第 4 章](../chapter04_numpy_numerical_computation.md)

---

## 核心知识点

可以通过嵌套原生列表的形式来模拟多维矩阵结构。例如，包含数值的单行或单列一维数组可组合成表示 \(i \times j\) 元素的二维矩阵，甚至 \(i \times j \times k\) 等更高维度的多维立方体数据表示。

## 案例数据

```python
v = [0.5, 0.75, 1.0, 1.5, 2.0]
m = [v, v, v]
# [[0.5, 0.75, 1.0, 1.5, 2.0], ...] 共 3 行
```

## 易错细节：对象引用陷阱

若通过 `[v, v, v]` 创建矩阵 `m`，一旦修改共享子列表的首元素（如 `v[0] = 'Python'`），矩阵 `m` 的**所有行**的首个元素都会同步改变。原因是嵌套的三个子列表本质上指向**同一个对象的内存地址**（浅拷贝）。

## 解决方案

为避免数据互相污染，必须使用物理副本代替引用指针：

```python
from copy import deepcopy

m = 3 * [deepcopy(v), ]  # 每行独立内存
```
