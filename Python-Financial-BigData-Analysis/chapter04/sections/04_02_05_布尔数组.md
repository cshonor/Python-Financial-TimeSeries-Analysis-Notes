# 4.2.5 布尔数组

## 核心主旨

抛弃索引游标，利用并行化的布尔遮罩（Boolean Mask）直接定位过滤特征数据。

## 逻辑判定与生成

当编写 `h > 8` 时，逻辑运算符被广播渗透至每一项，返回一套与原数组等大的 `True/False` 矩阵。

```python
h = np.arange(10)
mask = h > 8
h[mask]  # 选取满足条件的元素
```

## 条件替代赋值：`np.where`

```python
np.where(h % 2 == 0, 'even', 'odd')
```

- 条件为 `True` 落位 `'even'`，为 `False` 落位 `'odd'`
- 一句代码取代多级 `for + if-else` 循环
