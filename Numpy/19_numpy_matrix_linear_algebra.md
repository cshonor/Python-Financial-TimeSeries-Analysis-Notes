# 19 线性代数极简入门（机器学习打底，够用就行）

> 目标：不推公式，只掌握 ML/量化最常见的矩阵操作：点积、矩阵乘、转置、求逆、解线性方程、特征值/特征向量的直觉。

---

## 1. 向量与矩阵的形状（shape 是一切的源头）

- 向量：`(n,)`
- 行向量：`(1, n)`
- 列向量：`(n, 1)`

很多“报错”本质是 shape 不匹配。

---

## 2. 点积与矩阵乘法（`@`）

- 点积：两个向量的内积，输出标量  
  - `a @ b`（当 `a,b` 都是 1D）
- 矩阵乘：线性变换/特征组合的核心  
  - `X @ w`

量化/ML 常见：

- `X`：特征矩阵（样本×特征）
- `w`：权重向量（特征×1）
- `y = X @ w`：线性模型输出（回归/打分）

---

## 3. 转置（`.T`）

- `X.T`：把行列互换
- 常见用途：
  - 构造 \(X^T X\)
  - 相关/协方差计算的中间步骤

---

## 4. 逆矩阵与“别乱求逆”

- `np.linalg.inv(A)`：求逆（要求 `A` 可逆）
- 实务建议：
  - 能用 `np.linalg.solve(A, b)` 就别 `inv(A) @ b`
  - `solve` 更稳定、更快

---

## 5. 解线性方程（推荐 `solve`）

解 \(Ax=b\)：

```python
x = np.linalg.solve(A, b)
```

这在：

- 线性回归的正规方程（概念层面）
- 组合优化的一些简化场景

都很常见。

---

## 6. 特征值/特征向量（直觉）

- `np.linalg.eig(A)`：一般矩阵
- `np.linalg.eigh(A)`：对称矩阵（更常见：协方差矩阵）

直觉：

- 特征向量：某个“方向”
- 特征值：沿该方向的“伸缩比例”

在 ML 里最常见的落地就是 PCA（主成分分析）：协方差矩阵的前几个特征向量给出“最重要的方向”。

---

## 7. 最小够用的 NumPy API 列表

- `@`：矩阵乘法
- `.T`：转置
- `np.dot` / `np.matmul`（可用，但更推荐 `@`）
- `np.linalg.solve`
- `np.linalg.inv`（慎用）
- `np.linalg.eig` / `np.linalg.eigh`

---

## 8. 一个量化/ML 直观小例子（思想，不追完美）

- 你有 3 个特征：动量、波动率、成交量因子
- 给一个权重向量 `w`
- 用 `score = X @ w` 给每个样本打分
- 再按 score 排序做选股/分层

这就是“线性代数在策略里的最常见形态”。
